# ⚡️ PokeBuildMaker (ポケビルドメーカー)

ポケモン育成論（ビルド）を美しく、直感的に作成・共有できるWebツールです。  
PokeAPIを活用し、最新の全ポケモン・技・道具のデータを日本語で検索・反映できます。

## ✨ 主な機能

- **全ポケモン対応**: フォルムチェンジやメガシンカ、最新作（SV）のポケモンまで網羅
- **日本語検索**: ポケモン名、技名、持ち物をすべて日本語で検索可能
- **自動ステータス計算**: 努力値・個体値・性格補正をリアルタイムに反映し、Lv50/Lv100の実数値を自動算出
- **高画質画像出力**: 生成したカードを1280x720の美麗なPNG画像として保存可能
- **レスポンシブ対応**: PC、タブレット、スマホのどのデバイスからでも作成可能
- **モダンなUI/UX**: ダークモードを基調とした、直感的でプレミアムな操作感

## 🚀 公開（デプロイ）について

このプロジェクトは **HTML/CSS/JSのみの静的サイト** ですので、GitHub Pages などで非常に簡単に（かつ無料で）公開できます。

1. **GitHub Pages (おすすめ)**:
   - GitHubにリポジトリを作成してプッシュした後、リポジトリの `Settings` > `Pages` で `main` ブランチを選択するだけで公開完了です。

## 🛠 使用技術

- **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **API**: [PokeAPI](https://pokeapi.co/)
- **Library**: `html2canvas` (画像生成用)

## 📝 使い方
1. ポケモンを選択（日本語で検索可能）
2. 努力値（ベース）、技、持ち物、性格などを設定
3. 右側のプレビューカードをリアルタイムに確認
4. プレビュー画像を右クリック（スマホは長押し）して保存し、SNSやブログに貼り付け！
